# Notes Manager CLI

This project is a simple command-line Python application that allows users to manage personal notes.  
The project was built step-by-step in multiple versions to understand how real software evolves.  
It also includes a practice of using **GitHub pull requests (PRs)** by improving the project in different versions.

---

## How It Works

- The program runs in a loop (menu system)
- User selects options to add, view, delete, or search notes
- Notes are stored in a simple text file (`notes.txt`)
- Each version improves functionality and structure

---

## Features

- Add new notes
- View all saved notes
- Delete all notes (reset file)
- Search notes by keyword
- Simple CLI menu system

---

## Versions

| Version | Description |
|----------|------------|
| V1 | Basic CLI notes app with add and view functionality |
| V2 | Added delete and search features + improved menu |
| (Future) | Plan to improve structure and maybe add single-note deletion |

---

## Pull Request Practice

This project is intentionally structured to simulate real development workflow:

- Version 1 was the initial working application
- Version 2 was created in a separate branch
- A pull request was made to merge improvements into the main version

This helped me understand how developers:
- Work on features separately
- Improve code step-by-step
- Use GitHub PRs to manage changes safely
